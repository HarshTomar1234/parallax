Repository detailing Computer Vision (CV) model and framework experimentation.

### Core Disciplines
*   Image Processing:
    *   Filtering
    *   Edge Detection
    *   Feature Extraction (SIFT, SURF, ORB)
*   Machine Learning for Vision:
    *   Classical CV algorithms
    *   Statistical pattern recognition
*   Deep Learning for Vision:
    *   Convolutional Neural Networks (CNNs)
    *   Recurrent Neural Networks (RNNs) for video analysis
    *   Vision Transformers

### Key Tasks
*   **Image Classification**: Categorization of entire images.
*   **Object Detection**: Localizing and classifying multiple objects within an image (bounding boxes).
*   **Semantic Segmentation**: Pixel-level classification for scene understanding.
*   **Instance Segmentation**: Pixel-level classification and individual instance differentiation.
*   **Pose Estimation**: Identifying key points and orientation of objects or humans.
*   **Generative Models**: Image synthesis and manipulation (GANs, VAEs, Diffusion Models).

### Frameworks and Libraries
*   **TensorFlow**: End-to-end open source machine learning platform.
*   **PyTorch**: Open source machine learning framework, emphasizing flexibility.
*   **OpenCV**: Comprehensive library for real-time computer vision.
*   **Keras**: High-level neural networks API, typically running on TensorFlow.
*   **Scikit-image**: Image processing library for Python.

### Model Architectures
*   **CNN Variants**: LeNet, AlexNet, VGG, ResNet, Inception.
*   **Object Detection**:
    *   R-CNN family (Faster R-CNN, Mask R-CNN)
    *   Single-shot detectors (YOLO, SSD)
*   **Segmentation**: U-Net, DeepLab.
*   **Attention-based**: Vision Transformers (ViT), DETR.
