# Machine-and-Deep-Learning-NLP Learnings

## Repository Scope
Collection of applied studies and conceptual implementations across machine learning, deep learning, and natural language processing domains. Focus on algorithmic understanding and practical application.

## Core Disciplines

### Machine Learning
*   **Supervised Learning**:
    *   Classification: Logistic Regression, Support Vector Machines (SVMs), Decision Trees, Random Forests, Gradient Boosting Machines (GBM), XGBoost.
    *   Regression: Linear Regression, Polynomial Regression, Ridge, Lasso, Elastic Net.
*   **Unsupervised Learning**:
    *   Clustering: K-Means, DBSCAN, Hierarchical Clustering.
    *   Dimensionality Reduction: Principal Component Analysis (PCA), t-Distributed Stochastic Neighbor Embedding (t-SNE), Independent Component Analysis (ICA).
*   **Model Evaluation**: Cross-validation strategies, performance metrics (accuracy, precision, recall, F1-score, ROC-AUC, RMSE, R-squared), hyperparameter tuning (Grid Search, Random Search, Bayesian Optimization).
*   **Feature Engineering**: Data scaling, encoding categorical variables, feature selection techniques.

### Deep Learning
*   **Neural Network Architectures**:
    *   Feedforward Neural Networks (FNNs) for tabular and basic classification/regression.
    *   Convolutional Neural Networks (CNNs) for image recognition and computer vision tasks.
    *   Recurrent Neural Networks (RNNs), Long Short-Term Memory (LSTM) networks, Gated Recurrent Units (GRUs) for sequential data processing.
    *   Transformer architectures for advanced sequence modeling and attention mechanisms.
*   **Fundamentals**: Activation functions (ReLU, Sigmoid, Tanh, Softmax), loss functions (MSE, Cross-Entropy), optimizers (SGD, Adam, RMSprop, Adagrad), backpropagation, regularization (L1, L2, Dropout, Batch Normalization).
*   **Frameworks**: TensorFlow, PyTorch.

### Natural Language Processing (NLP)
*   **Text Preprocessing**: Tokenization (word, subword), stemming, lemmatization, stop word removal, text normalization, part-of-speech tagging.
*   **Feature Representation**:
    *   Bag-of-Words (BoW), TF-IDF vectorization.
    *   Static Word Embeddings: Word2Vec, GloVe, FastText.
    *   Contextual Embeddings: BERT, GPT, ELMo.
*   **NLP Tasks**:
    *   Text Classification (Sentiment Analysis, Spam Detection, Topic Modeling).
    *   Named Entity Recognition (NER), Part-of-Speech Tagging.
    *   Machine Translation, Sequence-to-Sequence models.
    *   Text Generation, Summarization.
    *   Question Answering, Semantic Search.
*   **Models**: Naive Bayes, SVMs for text, Hidden Markov Models (HMMs), Conditional Random Fields (CRFs), Sequence models (RNNs/LSTMs), Transformer-based models.
